## 使用说明
1. 安装编译
```shell script
# 拉取代码
git clone git@github.com:openstack-test/kubespace.git

# 打包
cd kubespace
go build main.go -o ./server

# 启动
./server

# 启动前端
cd kubespace/kubespace_fe
npm run dev
```

2. 启动服务前先创建etc/config.yaml
```shell script
# 数据库配置
mysql:
  path: '127.0.0.1:3306'
  db-name: 'kubespace'
  username: 'root'
  password: '123456'
```

3. 初始化数据库
```go
windows执行以下脚本, 初始化数据库

init_db.bat
```
